Bạn chỉ cần copy dòng code siêu ngắn này dán vào bất kỳ trang web nào bạn muốn hiển thị:

<script src="[https://d4m-dev.github.io/WidgetMusic.com/music-loader.js](https://d4m-dev.github.io/WidgetMusic.com/music-loader.js)"></script>


Mẹo nhỏ:

Nếu bạn không đổi tên file thành index.html mà vẫn để là music_widget.html, thì bạn sửa code nhúng một chút như sau:

<script 
    src="[https://d4m-dev.github.io/WidgetMusic.com/music-loader.js](https://d4m-dev.github.io/WidgetMusic.com/music-loader.js)" 
    data-source="[https://d4m-dev.github.io/WidgetMusic.com/music_widget.html](https://d4m-dev.github.io/WidgetMusic.com/music_widget.html)">
</script>


Widget sẽ tự động hiển thị với chiều rộng tối đa 400px (giống giao diện điện thoại) khi xem trên máy tính, và tự động full màn hình khi xem trên điện thoại.
